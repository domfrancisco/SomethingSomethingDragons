# SomethingSomethingDragons

Basic reactive web page starter using vanilla HTML, CSS, and JavaScript.

## Run locally

Open `index.html` directly in your browser, or serve the folder with any static file server.

## Structure

- `index.html` - app markup and script/style links
- `styles/main.css` - styling, layout, and animations
- `scripts/main.js` - state and reactive UI rendering
